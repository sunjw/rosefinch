jquery.common

Introduction

jquery.common provides some basic and common parts to developers.

You can use it to put some placeholding text in input box and to verify user's input before submit the form. it can check if there is nothing in input box or if the text is an email address or number. Also it can check custom input by providing a regular expression.

All selector used in functions can be used as jQuery selector. Like this: jqCommon.setPlaceholder("#aform", "#something1", "Type something here..."); means jqCommon will put "Type something here..." into $("form#aform input#something1") as a placeholding string.

jqCommon.stopBubble(e)

A function will stop the event bubbles up.

When you move mouse on an element, there will be an event bubbling up through DOM tree. Sometimes you don't want it to bubble up to another element, you need to use this function to stop it.

Use it like this:

<div id="uplevelElem">
    <div id="someElem">
    
    </div>
</div>
<script type="text/javascript">
var elem = $("#uplevelElem #someElement");
elem.mouseover(function (e) {
        someHandlerFunction();
        jqCommon.stopBubble(e);
    });
</script>

After calling jqCommon.stopBubble(e), the event will never go up to <div id="uplevelElem"></div>.

jqCommon.setPlaceholder(formSelector, inputSelector, text)

A function will put some placeholding string into specified input box.

formSelector is a selector of the form which contains the input element. You can use null, then jqCommon will find the input in body element.

inputSelector is a selector of the input which will have the placeholding string. This is acturally use $("form" + formSelector).find("input" + inputSelector) to find the input element.

Use it like this:

<form id="aform">
    <input id="something1" type="text" />
</form>
<script type="text/javascript">
jqCommon.setPlaceholder("#aform", "#something1", "Type something here...");
</script>

jqCommon.setVerify(formSelector, inputSelector, type, okHandler, errHandler)

A function will make verification in specified input box when focus is lost and form is submitting.

formSelector is a selector of the form which contains the input element.

inputSelector is a selector of the input which will be verified.

type tells jqCommon how to verify the input. It now supports "empty", "email", "number" and "custom". "empty" means the input cannot be empty. "email" means the input can only take an email address. "number" means the input can only take number. "custom" means the input will be verified by specified regular expression. You should set the regular expression by calling jqCommon.setCustomRegex(formSelector, inputSelector, regex).

okHandler is a function to be called if verification is passed. It can be null.

errHandler is a function to be called if verification is NOT passed. It can be null.

Use it like this:

<form id="aform">
    <input id="emptyInput" type="text" />
    <input id="emailInput" type="text" />
    <input id="numberInput" type="text" />
    <input id="customInput" type="text" />
</form>
<script type="text/javascript">
jqCommon.setVerify("#aform", "#emptyInput", "empty", function(){alert("right!")}, function(){alert("wrong!")});
jqCommon.setVerify("#aform", "#emailInput", "email", null, function(){alert("wrong!")});
jqCommon.setVerify("#aform", "#numberInput", "number", function(){alert("right!")}, null);
jqCommon.setCustomRegex("#aform", "#customInput", /^[a-zA-Z0-9]{2,20}$/);
jqCommon.setVerify("#aform", "#customInput", "custom", function(){alert("right!")}, null);
</script>

jqCommon.setFormSubmitHandler(handler)

Give a handler function to jqCommon. After form verification, the function will be called to handle result and form submitting.

handler is a function. It should be defined as function (formObj, result) {...}. When this function is called, it will receive to parameters: formObj and result. formObj is a jquery object contains the form which just be verified. result is result of verification. It will be true or false, which means the form is passed verification or not.

Use it like this:

<form id="aform">
    <input type="text" name="something" />
    <div class="formCheck"></div>
</form>
<form id="bform">
    <input type="text" name="something" />
    <div class="formCheck"></div>
</form>
<script type="text/javascript">
jqCommon.setFormSubmitHandler(function (formObj, result) {
        if(!result) {
            // Not passed
            if(formObj.attr("id") == "aform")
                formObj.find(".formCheck").html("a form somthing wrong");
            else if(formObj.attr("id") == "bform")
                formObj.find(".formCheck").html("b form somthing wrong");
        }

        return result;
    });
</script>
